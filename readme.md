# Cara Menjalankan

## Cara Menjalankan Aplikasi Client (Frontend)

- Setting .env

```bash
cd client
npm install
npm run dev
```

## Cara Menjalankan Aplikasi Server (Backend)

- Nyalaka Mysql dan buat database dengan nama "peminjaman-buku"
- Setting .env

```bash
cd server
npx prisma migrate
npm install
node index.js
```

`jalankan secara bersamaan ya :)`
